1) Usé el comando <section id =""> para poder hacer hipervínculos de ancla, para poder desplazarse a traves de la página vinculado a la etiqueta <a href= "">.

2) Usé el comando <article> porque esta función sirve para separar artículos dentro de una misma sección, siendo ellos mismos artículos quese podrían distribuir por si mismos. Un ejemplo serían revistas, una tarjetas de producto, etc.

3) El comando <aside> muestra contenido relacionado con la sección pero secundario en comparación al artículo principal de la sección. Ejemplo: Una barra lateral, anuncios, datos curiosos, etc.
